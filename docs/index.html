    color-index

    @function color-index($number) { ... }

    Description

    Color Index Converts a value from a numerical index to a usable percentage

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $number

    Must be a positive integer < 1000 & divisible by 50

    		Number(unitless)— none

    Returns

    Number(%)

    Example

    .color-index(400) -> 20%

    500 is 0. every 50 represents 10%.
Inputs less than 500 will be converted to positive percentages,
Inputs greater than 500 will be converted to negative percentages.

    Throws

    • Invalid arg $number, expected a positive integer less than 1000 that is also divisible by 50. $number: #{error.inspect-param($number)}

    Requires

    • [external] v.is-integer — (validation.scss)
    • [external] error.inspect-param — (error.scss)
